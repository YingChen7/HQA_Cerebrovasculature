To plot the locally estimated scatterplot smoothing (LOESS) curves

1. Download the 4_R_LOESS_curve foloder to local device;
2. See hierarchical cortical and vascular volumes of each individual that are saved in csv files in the folder 
   PATH_you_save/4_R_LOESS_curve/data;
3. In every R script, set the variable dir_ as PATH_you_save/4_R_LOESS_curve;
4. Run R scripts.
